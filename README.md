# What’s Friday — Coming Soon Landing Page

Static landing page for What’s Friday. Focused on a clean, responsive design with quick contact and early access email.

## Preview

- Open `index.html` directly in your browser, or
- Serve locally (any static server works). Examples:
  - Python: `python -m http.server 8080`
  - Node (if installed): `npx serve -l 8080`

Then visit `http://localhost:8080`.

## Customization

- Title, description, and social meta: `index.html`
- Colors and layout: `styles.css`
- JS enhancements: `script.js`
- Favicon/mark: `assets/favicon.svg`
